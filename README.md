* Function<br>
　Function 関数名(引数) { 処理 return(戻り値) }<br>
 <br>
呼び出しは以下の4通り<br>
・引数なし/戻り値なし　→　関数名<br>
・引数なし/戻り値あり　→　$変数名 = 関数名<br>
・引数あり/戻り値なし　→　関数名 引数1 引数2　※スペース区切りで表記<br>
・引数あり/戻り値あり　→　$変数名 = 関数名 引数1 引数2<br>
